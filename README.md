## **Portfolio**

![image](https://github.com/gsoaresdz/portfolio/assets/69989654/d696fb23-20ec-4adc-b9ab-caaaf20ef89c)

### **Descrição**

Este projeto consiste em um site de portfólio pessoal, projetado para demonstrar habilidades, projetos e experiências. Ideal para profissionais que desejam marcar presença online, o site é estático, facilitando a hospedagem e a manutenção.

### **Estrutura do Projeto**

O projeto é organizado da seguinte forma:

- **`css/`**: Pasta contendo arquivos de estilo CSS para personalização visual do site.
- **`img/`**: Pasta com imagens utilizadas no site, incluindo projetos, fotografias pessoais ou ícones.
- **`favicon.svg`**: Ícone de favorito que aparece nas abas do navegador.
- **`index.html`**: Arquivo HTML principal, estrutura a base do site e conteúdo.

### **Requisitos**

Para visualizar e editar este projeto, você precisará de:

- **Navegador Web**: Qualquer navegador moderno, como Chrome, Firefox, Safari, ou Edge.
- **Editor de Código**: Recomenda-se o Visual Studio Code (VSCode) para edição e gerenciamento do projeto.

### Visual Studio Code (VSCode)

- **Versão Requerida**: Recomenda-se a última versão estável do VSCode para garantir compatibilidade com as extensões e ferramentas mais recentes.
- **Download e Instalação**: Visite Visual Studio Code para baixar e instalar a última versão.

### **Configuração do Ambiente**

1. **Instalação do VSCode**: Siga as instruções de instalação no site oficial do VSCode para seu sistema operacional.
2. **Extensões Recomendadas**:Você pode instalar essas extensões diretamente pelo VSCode, acessando a aba de extensões (**`Ctrl+Shift+X`** ou **`Cmd+Shift+X`** no macOS) e buscando por seus nomes.
    - **Live Server**: Facilita a visualização de páginas web em tempo real, com recarga automática ao editar arquivos.
    - **Prettier**: Formata o código automaticamente para melhor legibilidade e consistência.

### **Visualização e Edição**

Para visualizar e editar o site:

1. **Abra o Projeto no VSCode**: Utilize o menu **`File > Open Folder`** e selecione a pasta do projeto.
2. **Visualização ao Vivo**: Com o Live Server, clique com o botão direito no arquivo **`index.html`** e selecione **`Open with Live Server`** para ver o site no navegador com atualizações em tempo real.
